#Introduction

Sketch Word: 나만 모르는 말풍선

플레이어는 자신의 휴대폰을 머리 위에 올려 단어를 볼 수 없고 오직 상대방의 성대모사, 연기, 노래, 마임 혹은 설명을 통해 여러 단어들을 정해진 시간 내 맞추는 게임이다.

#Numerous Contets

스무가지의 주제와 천 여개의

최신유행의 다양한 주제들을 20개 이상 제공하고 즐겨찾기를 통해 손 쉽게 좋아하는 주제 선택이 가능하다.

#Custom Contents

나만의 컨텐츠를 만들다

기본 제공되는 다양한 주제들에서 추가로 10개의 자신만의 단어들을 통해 나만의 주제를 만들 수 있다

#Game Interface

터치 / 모션 인터페이스 지원

터치 인터페이스 우측 = CORRECT
터치 인터페이스 좌측 = PASS

모션 인터페이스 Tilt Down = CORRECT
모션 인터페이스 Tilt Up = PASS

#Preparing Section

게임시작 전 인터페이스 설명

선택한 게임 인터페이스를 준비할 수 있는 섹션을 제공한다. 이때 자이로 센서가 지면과 수직을 이뤄야만 게임이 실행되며, 터치 인터페이스를 선택한 경우 시작버튼을 클릭해야 게임을 실행할 수 있다.

Play "Sketch Word"

맞은편의 사람은 단어를 설명한다.

#Final Score

최종 스코어를 보여주는 페이지

최대 5명의 플레이어들의 점수를 나열하여 최종 승자를 보여주는 페이지이다. 이 화면에서 다시하기 버튼을 누르면 다른 주제를 선택할 수 있으며, 새로 시작할 경우 플레이어 수와 시간을 재설정할 수 있다.
